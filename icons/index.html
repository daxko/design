      <Icon name="chevron-right" />
    );
  }

Html

If youâ€™re not using React, you will manually need to include the icon svg set. CSS Tricks provides some best practice tips for including SVG icons.

  <use xmlns:xlink="http://www.w3.org/1999/xlink" xlink:href="#icon-barcode"></use>

Styling the Icons

The icons can be styled naturally through CSS.

[class*=icon-] {
  stroke: #000;
  fill: #000;
  width: 48px;
  height: 48px;
}
